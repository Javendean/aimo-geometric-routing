# The Scaling Myth (Jan 22, 2026)
Source: https://richardaragon.substack.com/p/the-scaling-myth-why-modern-ai-is

## Core Argument
Scale without structure = memorization dead-ends. Modern AI systems hit walls not because they lack parameters but because they lack geometric organization.

## Implication for AIMO 3
gpt-oss-120B has 117B parameters (5.1B active via MoE). The model has MORE than enough capacity for IMO-level math. The reference bench shows GPT-5 Pro (which is larger but not dramatically so in active params) solves 10/10.

The gap between 4/10 and 10/10 is likely not about scale — it's about how the model's existing capacity is ORGANIZED for mathematical reasoning. This is exactly Aragon's thesis: structure > scale.
